stock_simulator/
│
├── main.py                ← 실행 시작점 (main loop)
├── requirements.txt       ← 의존 패키지 목록 (yfinance, pygame 등)
├── config/
│   └── layout.py          ← 화면 UI 배치 정보
│
├── core/
│   ├── game_state.py      ← 전역 게임 상태, 포트폴리오, 비교모드 등
│   ├── init.py            ← 데이터 초기화, 캐시, 다운로드 관리
│   └── data_loader.py     ← yfinance 데이터 로딩, 캐싱
│
├── ui/
│   ├── chart.py           ← 일반/확대/비교 차트 렌더링
│   ├── buttons.py         ← 버튼 위치 계산, 클릭 처리
│   ├── portfolio.py       ← 포트폴리오 출력 및 수익 계산
│   ├── alerts.py          ← 알림 표시
│   └── text.py            ← draw_text 등 공통 출력
│
├── save/
│   ├── save_load.py       ← 저장/불러오기 기능
│
├── assets/
│   ├── fonts/
│   └── images/            ← 아이콘, 그래프 배경 등
│
├── utils/
│   ├── date_utils.py      ← 날짜 관련 변환, 비교
│   └── constants.py       ← TICKERS, COMPANY_COLORS 등
│
└── cache/
    └── *.csv              ← 캐시된 주가 데이터

| 순위   | 기능                  | 설명 및 이유                             |
| ---- | ------------------- | ----------------------------------- |
| ✅ 1  | **뉴스 이벤트 추가**       | 지금 이미 있는 구조에 몇 개 더 넣으면 됨 (랜덤/수동)    |

| 🔴 7 | **PER, ROE 지표 표시**  | 재무제표 용어 학습 필요하므로 후순위 (선택사항)         |
